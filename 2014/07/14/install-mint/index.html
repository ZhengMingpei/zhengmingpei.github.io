安装Linux Mint 17后要做的配置

发表于 2014-07-14 更新于 2021-11-28 分类于 大学

2015/1/28更新

Linux Mint 17(Qiana)已经发布了几个月了，现在在distrowatch的排名已经蝉联第一很久了。

我本来是Ubuntu用户，但相对来说不太喜欢Unity和Gnome3，就转到了LinuxMint,安装了最新的Qiana（Cinnamon版本）。

从Ubuntu转过来，发现LinuxMint很多方便的地方，但也有一些需要配置的东东，其中比较实用的主要包括显卡驱动、WIFI驱动、中文输入法。

0.配置源

要用系统的更新管理器进行软件包更新，目前LinuxMint在国内的源较少，大学里使用的主要是中科大的源，更新划分为5个级别，这里我更新了前四个等级的160多MB。

1.显卡及无线网络的驱动

显卡的问题主要与Linux下的图形显示及游戏有关，我关于在linux下玩转Dota2的博客里曾有所提及。我的笔记本在系统使用官方的驱动管理器可以看到有三个选项，选择英伟达的官方驱动就好。

至于无线驱动，安装之后可以正常使用wifi联网，蓝牙我没有试过，不过以前在Ｕbuntu上，蓝牙效果不好。

2.中文输入法

安装fcitx，具体按照以下命令：

$ sudo apt-get install fcitx

还要安装fcitx-ui、fcitx-googlepinyin和fcitx-table之类，具体的按tab键然后补全选择即可。
